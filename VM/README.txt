vm.c основной файл VM
test.c упаковывает инструкции из массива program в файл sum.obj
для запуска необходимо собрать проект с помощью make и выполнить команду ./vm.out sum.obj

Пример программы для VM:
0011 0000 0010 0100 / 0x3024 / TRAP TRAP_GETUINT / считать данные из консоли в REG_N0
0000 0010 0010 0011 / 0x223 / ADD REG_N1, REG_N0, 0x3 / REG_N1 = REG_N0 + 0x3
0000 0000 0110 0000 / 0x60 / ADD REG_N0, REG_N1, 0x0 / REG_N0 = REG_N1 + 0x0
0011 0000 0010 0101 / 0x3025 / TRAP TRAP_PUTUINT / вывести значение из REG_N0
0011 0000 0010 0110 / 0x3026 / TRAP TRAP_HALT / завершение программы
